# 管理员访问指南

## 🔐 **管理员专用功能**

### **数据管理页面**
```
直接访问: http://localhost:3001/admin
```

### **数据状态监控**
```
直接访问: http://localhost:3001/status
```

### **命令行工具**
```bash
# 检查数据状态
node -r tsx/cjs scripts/check-data.ts

# 批量预抓取
node -r tsx/cjs scripts/batch-scrape.ts

# 实时进度监控
node -r tsx/cjs scripts/monitor-progress.ts
```

## 📊 **管理功能说明**

### **1. 数据管理页面** (`/admin`)
- 📈 **数据统计**: 查看30天数据覆盖情况
- 🔄 **批量更新**: 一键触发批量预抓取
- 📅 **详细状态**: 每日数据完整性检查
- 🛠️ **系统控制**: 管理员操作界面

### **2. 数据状态页面** (`/status`)
- 📊 **实时进度**: 自动刷新的进度监控
- 📈 **可视化**: 进度条和状态图表
- 📅 **日历视图**: 30天数据完整性一览
- ⏰ **自动更新**: 每30秒刷新数据

### **3. 命令行工具**
- 🔍 **数据检查**: 快速查看数据统计
- 🚀 **批量处理**: 后台批量预抓取
- 📺 **实时监控**: 控制台实时进度显示

## 🎯 **用户页面清理**

## 🔒 **安全建议**

### **生产环境部署**
1. **访问控制**: 为 `/admin` 路由添加身份验证
2. **IP限制**: 限制管理页面的访问IP
3. **HTTPS**: 使用安全连接访问管理功能
4. **日志记录**: 记录所有管理操作

### **开发环境**
- 管理员可直接访问所有功能
- 用户只能看到查询和状态功能
- 命令行工具仅限本地使用

现在用户界面更加简洁，管理功能完全分离！🎉
